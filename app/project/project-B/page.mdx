# Image Comparison Component Demo

This project demonstrates the usage of the ImageComparison component with a custom slider design. The component allows users to interactively compare two images by dragging a slider between them.

---

## Interactive Image Comparison

Below is an example of the ImageComparison component showing the difference between dark and light themes of Motion Primitives (click to view in full screen):

<MorphingImageDialog
  className="cursor-zoom-in"
  type="comparison"
  leftSrc="/mp_dark.png"
  leftAlt="Motion Primitives Dark"
  rightSrc="/mp_light.png"
  rightAlt="Motion Primitives Light"
  trigger={
    <ImageComparison className='aspect-16/10 w-full rounded-lg border border-zinc-200 dark:border-zinc-800'>
      <ImageComparisonImage
        src='/mp_dark.png'
        alt='Motion Primitives Dark'
        position='left'
      />
      <ImageComparisonImage
        src='/mp_light.png'
        alt='Motion Primitives Light'
        position='right'
      />
      <ImageComparisonSlider className='w-2 bg-white/50 backdrop-blur-xs transition-colors hover:bg-white/80'>
        <div className='absolute left-1/2 top-1/2 h-8 w-6 -translate-x-1/2 -translate-y-1/2 rounded-[4px] bg-white' />
      </ImageComparisonSlider>
    </ImageComparison>
  }
/>

## Component Features

The ImageComparison component includes:

- **Responsive Design**: Adapts to different screen sizes with the `aspect-16/10` ratio
- **Custom Styling**: Border styling that adapts to light/dark themes
- **Interactive Slider**: Smooth drag interaction with visual feedback
- **Accessibility**: Proper alt text for screen readers

The MorphingImageDialog component adds:

- **Modal Zoom**: Click to zoom images into a full-screen modal experience
- **Smooth Animations**: Morphing transitions powered by Framer Motion
- **Dual Support**: Works with both single images and image comparisons
- **Format Support**: Supports JPG, PNG, GIF and other web image formats
- **Keyboard Navigation**: ESC to close, Tab navigation support

### Component Structure

**Basic Image Comparison:**
```jsx
<ImageComparison className='aspect-16/10 w-full rounded-lg border border-zinc-200 dark:border-zinc-800'>
  <ImageComparisonImage
    src='/mp_dark.png'
    alt='Motion Primitives Dark'
    position='left'
  />
  <ImageComparisonImage
    src='/mp_light.png'
    alt='Motion Primitives Light'
    position='right'
  />
  <ImageComparisonSlider className='w-2 bg-white/50 backdrop-blur-xs transition-colors hover:bg-white/80'>
    <div className='absolute left-1/2 top-1/2 h-8 w-6 -translate-x-1/2 -translate-y-1/2 rounded-[4px] bg-white' />
  </ImageComparisonSlider>
</ImageComparison>
```

**Single Image with Morphing Dialog:**
```jsx
<MorphingImageDialog
  className="cursor-zoom-in"
  type="single"
  src="/image.jpg"
  alt="Description"
  trigger={
    <img src="/image.jpg" alt="Description" className="w-full rounded-lg" />
  }
/>
```

**Image Comparison with Morphing Dialog:**
```jsx
<MorphingImageDialog
  className="cursor-zoom-in"
  type="comparison"
  leftSrc="/image1.jpg"
  leftAlt="Image 1"
  rightSrc="/image2.jpg"
  rightAlt="Image 2"
  trigger={
    <ImageComparison className='aspect-16/10 w-full rounded-lg border border-zinc-200 dark:border-zinc-800'>
      <ImageComparisonImage src="/image1.jpg" alt="Image 1" position="left" />
      <ImageComparisonImage src="/image2.jpg" alt="Image 2" position="right" />
      <ImageComparisonSlider className='w-2 bg-white/50'>
        <div className='absolute left-1/2 top-1/2 h-8 w-6 -translate-x-1/2 -translate-y-1/2 rounded-[4px] bg-white' />
      </ImageComparisonSlider>
    </ImageComparison>
  }
/>
```

## Usage Notes

**ImageComparison Component:**
- The `ImageComparisonImage` components require `src`, `alt`, and `position` props
- The `ImageComparisonSlider` can be customized with additional styling
- The component supports both light and dark theme variations
- Images are loaded from the `/public` directory

**MorphingImageDialog Component:**
- Wraps any image content to make it clickable and zoomable
- Supports two types: `"single"` for individual images, `"comparison"` for image comparisons
- Add `cursor-zoom-in` className to the trigger for better UX
- Modal content is rendered in a portal and includes backdrop blur
- Automatically handles keyboard navigation (ESC to close)
- Click outside the modal to close

## Customization Options

You can customize the slider appearance by modifying:

- **Width**: Adjust the `w-2` class for slider thickness
- **Background**: Change `bg-white/50` for different opacity levels
- **Hover Effects**: Modify `hover:bg-white/80` for interaction feedback
- **Handle Design**: Customize the inner div for different handle styles
